* Architecture
Modular config loaded via =config-loader.el=. All modules in =lisp/=, order matters:
1. Foundation: =path-utils= → =core-init= (GC, Iosevka fonts, transparency) → =theme-config= (doom-dracula faces)
2. Org: =org-basic= (PARA) → =org-capture-config= → =org-agenda-config= (⚡ views) → =org-roam-config=
3. Features: =gtd= (habits) → =reading-tracker= → =journal= → =dashboard= (startup screen)
4. Integrations: =zotero-bibtex= → =magit-config= → =spell-checking-config= (Jinx) → =latex-config= → =matlab-config= → =copilot-config=
5. UI: =modeline-config= → =keybindings=

* Setup (macOS)
#+begin_src sh
brew install enchant pkg-config
#+end_src
First launch auto-installs packages. Requires =~/org/= with PARA folders (auto-created).
